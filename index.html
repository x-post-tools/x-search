📖 GUIDE
1
下の各セクションで検索条件を入力する。入力は好きな項目だけでOK、全部埋める必要はなし
2
「生成されるクエリ」に自動で検索文字列が作られるのを確認する
3
「X で検索する」ボタンを押すと、Xの検索結果が新しいタブで開く
自分の過去ツイートを探すなら「ユーザー」の from: に自分のIDを入れて日付で絞るのが基本
バズったツイートを見つけたいなら「いいね数」に数値を入れるだけでOK
クエリをコピーしてXの検索バーに貼り付けて使うこともできる
🎲 ランダム検索 — FROM欄にIDを入れてボタンを押すと、過去のランダムな1ヶ月のツイートが表示される。何が出るかお楽しみ！
🕰 初ツイート探し — 登録年月を入力して月ごとに検索。登録年月はXプロフィール画面の「アカウントについて」で確認できる。0件なら「次の月 →」で進めばOK！

🎲 ランダム検索
下の「ユーザー」欄の from: にXのユーザーIDを入れてからボタンを押すと、過去のランダムな、いつの年月かわからない！ランダムに選択された1ヶ月のツイートが表示されます。何が出るかはお楽しみ！
🕰 初ツイート探し ※過去の自分のブログとしても活用できます。
登録年月から1ヶ月ずつ絞り込んで初ツイートを探せます。
💡 登録年月はXのプロフィール →「アカウントについて」で確認できます。
✨ 有効活用方法 ✨
その①　自分の過去をチェックワンツー！
初ツイート探しは「あの頃の自分」を掘り起こす過去ブログ感覚でも使えます。
懐かしの年月を入力して、当時の自分のつぶやきを振り返ってしみじみしてみてください！

その②　気になるあの人はあの時なにを！？
あの頃〇〇さんは何をつぶやいてたのかしら！？見てみたい！→IDに見てみたい方のIDを入力し、気になる年月をセレクトして検索！こんなことをツイートしてたのね～つまり全Twitter民のブログ状態が読めます。最高か！
（検索結果の表示順：月末→月初の順に表示されます）

例：※初ツイート探しの年月でセレクトして検索してください。
下記の大きな価格変動のあった「日経平均株価の月間最安値の日」を記載しています。大きく下げた日の一覧ではありませんが、1ヶ月単位で検索結果が表示されますので、その1ヶ月間のツイートを振り返りにどうぞ。という参考例です。
2009年3月10日
【終値】7,054.98円 （前日比 -31.05円 / -0.44%） 【解説】バブル後最安値。リーマンショック後の底打ち局面。ここから長期上昇トレンドへ。
2012年11月13日
【終値】8,661.05円 （前日比 -15.39円 / -0.18%） 【解説】アベノミクス前夜。14日の野田首相の解散発言を機に大底から反転急上昇へ。
2013年5月1日
【終値】13,799.35円 （前日比 -61.51円 / -0.44%） 【解説】5.23ショック。23日に大暴落（-1,143円）したが、それまでの急騰で結果的に月初が月間安値に。
2015年8月25日
【終値】17,806.70円 （前日比 -733.98円 / -3.96%） 【解説】チャイナショック。中国株暴落が世界に波及し、連日の暴落でこの日が底に。
2016年6月24日
【終値】14,952.02円 （前日比 -1,286.33円 / -7.92%） 【解説】ブレグジット。英国のEU離脱決定でパニック。この日がそのまま月間安値に。
2020年3月19日
【終値】16,552.83円 （前日比 -173.72円 / -1.04%） 【解説】コロナショック。パンデミックの恐怖で連日急落。19日に大底を形成。
2024年8月5日
歴代下落幅1位
【終値】31,458.42円 （前日比 -4,451.28円 / -12.40%） 【解説】歴史的大乱高下。米リセッション懸念等で日経平均下げ幅歴代1位を記録。
2024年8月6日
歴代上昇幅1位
【終値】34,675.46円 （前日比 +3,217.04円 / +10.23%） 【解説】日経平均上げ幅歴代1位。前日の歴史的暴落から一転、買い戻しが殺到し過去最大の急騰。
※残念ながらリーマンショックの2008年10月はTwitterがまだ日本で普及しておらず、当時のツイートはほぼ残っていません。
